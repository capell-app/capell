# Core Structural Spine Artwork Revision Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Replace the Core hero and marketplace card with a calm, authored architectural-spine composition that makes Capell's Laravel boundary, Core relationships, controlled extension, and application-owned frontend legible at full and thumbnail sizes.

**Architecture:** Nano Banana supplies only a text-free warm-paper/navy engraved environment. A deterministic SVG renderer owns the real Capell wordmark, all copy, semantic diagrams, boundaries, connectors, and distinct site outputs, then rasterizes exact progressive-sRGB JPEGs. Hero and card are independently composed so the compact marketplace card is not a crop of the hero.

**Tech Stack:** Nano Banana Pro, Node.js, SVG, `rsvg-convert`, ImageMagick, JPEG metadata inspection, focused PHPUnit/Pest and documentation gates.
