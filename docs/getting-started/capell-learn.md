# Capell Learn

![Capell Learn screenshot](../images/capell-readme-banner.jpg)

Capell Learn is the quickest path through the core concepts behind Capell. Start with the model relationships, then move into page authoring and extension points.

## Core Model Relationships

Start with the relationship maps when you need to understand how the main records connect before reading code.

- [How Capell works](how-capell-works.md) explains the model in prose with a compact visual overview.
- [Core relationship map](../reference/core-erd.md) shows the fuller delivery, asset, access, and operations relationships.
- [Blueprints](types.md) explains how blueprints shape page, widget, layout, and package behaviour.
